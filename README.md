# Capstone-Anti-Corruption-TTF

Here we make our pre final year project

# How to Run Backend
- npm i
- docker compose up
- npx prisma migrate dev
- npx prisma generate
- npx prisma studio

# How to Run Mobile Frontend
- npm i
- npx expo start
- a (for android)
- i (for ios)
